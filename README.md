# NightmareSurvival-Clone
Unity 2017 Nightmare shooting survival tutorial 클론 리포지토리

## 최초 설정
- Unity 버전 관련
	- 2021.3.29f1 로는 문제가 없음
	- 2022.3.7f1 는 바닥 머티리얼 문제 생김
